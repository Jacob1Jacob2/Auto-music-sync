*Originally based on the abandoned mod by xanii, fully updated and optimized for the modern Geode loader.*

### Settings
- **Preserve pitch**: keeps music pitch at normal while gameplay speed changes.
- **Portal buffer**: adjusts portal detection offset in pixels.
