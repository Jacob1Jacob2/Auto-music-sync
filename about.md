# Auto Music Sync

Automatically syncs the level's music to your player's speed! 

Whenever you miss a speed portal, the music will speed up or slow down to match where you're supposed to be. Disable the mod if you're playing a level uses speed portals as obstacles

### Features
* **Works Automatically:** Reads the level data to find speed portals natively.
* **Crash Free:** Completely rewritten from the ground up to fix the memory leak and string-parsing crashes present in older versions of this mod.
* **Optimized:** Uses direct engine hooks for zero lag when loading massive levels.

*Originally based on the abandoned mod by xanii, fully updated and optimized for the modern Geode loader.*
